# Craft Verse 🎨

Full-stack creative platform — DIY kits, custom projects, 3D printing, custom clothes.

## Quick Start

```bash
npm install
# create .env.local with MONGODB_URI + JWT_SECRET
npm run dev
# seed: GET /api/seed?key=YOUR_SEED_SECRET
```

## Deploy to Vercel
1. Push to GitHub
2. Import on vercel.com
3. Add env variables (MONGODB_URI, JWT_SECRET, CLOUDINARY_*, SEED_SECRET)
4. Deploy → visit /api/seed?key=SECRET to populate DB
